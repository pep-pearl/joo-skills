# Root Map

## Scope

Use for vague product/design requests in the React/FSD monorepo example.

## Top-Level Areas

- `apps/web`: main React app
- `packages/domains`: shared domain packages
- `packages/ui`: shared UI kit

## Available Map Shards

- `.ai/indexing/maps/routes.md`: route/page/screen work
- `.ai/indexing/maps/api.md`: API/query/client work
- `.ai/indexing/maps/state.md`: store/cache/session work
- `.ai/indexing/maps/domains/auth.md`: login/signup/session
- `.ai/indexing/maps/domains/dashboard.md`: dashboard pages/widgets

## Read Rule

Pick one shard next. Do not read all shards.
